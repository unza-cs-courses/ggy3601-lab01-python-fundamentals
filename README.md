## Purpose

This lab ensures you have a working Python environment and can write basic programs using variables, expressions, and print statements.

## Learning Outcomes

By completing this lab, you will be able to:
- LO1.1: Install and configure a Python development environment
- LO1.2: Create, save, and execute Python scripts
- LO1.3: Use variables to store different data types (int, float, str, bool)
- LO1.4: Write arithmetic expressions and use built-in functions
- LO1.5: Use print() with formatted output
